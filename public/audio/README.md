# Audio Files for ClarpSaasPromo

Drop these `.mp3` files into this folder before rendering.

## Required Files

| File | Type | Duration | Description |
|------|------|----------|-------------|
| `bg-music.mp3` | Music | 17s+ | Dark electronic/trap beat. Plays full video with fade in/out. Keep it low-key, not overpowering. |
| `sfx-type.mp3` | SFX | ~0.8s | Keyboard typing clicks (rapid). Plays during search bar typing. |
| `sfx-scan-start.mp3` | SFX | ~0.5s | Woosh/launch sound. Plays when scan is initiated. |
| `sfx-step-tick.mp3` | SFX | ~0.3s | Subtle tick/ping. Plays when each scan step completes (x5). |
| `sfx-scan-complete.mp3` | SFX | ~1s | Success chime/ding. Plays when full scan finishes. |
| `sfx-whoosh.mp3` | SFX | ~0.4s | Quick whoosh. Plays on scene transitions (x3). |
| `sfx-score-reveal.mp3` | SFX | ~1s | Rising tone/sweep. Plays during trust score count-up. |
| `sfx-cta-hit.mp3` | SFX | ~1s | Impact/bass drop. Plays at CTA closing scene entrance. |

## Specs

- Format: MP3 (44.1kHz, stereo preferred)
- Loudness: Normalize SFX to -12dB, music to -18dB (will be mixed in composition)

## Free Sources

- **Pixabay** — pixabay.com/sound-effects (royalty-free, no attribution)
- **Freesound** — freesound.org (CC0 filters available)
- **Mixkit** — mixkit.co/free-sound-effects (royalty-free)
- **Uppbeat** — uppbeat.io (free tier with attribution)
